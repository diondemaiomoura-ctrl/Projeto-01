# Projeto-01 — Lista de Tarefas

Aplicação web simples de lista de tarefas (_To-Do List_) desenvolvida com HTML, CSS e JavaScript puro.

## Funcionalidades

- ✅ Adicionar novas tarefas
- 🔄 Marcar/desmarcar tarefas como concluídas
- 🗑️ Remover tarefas individualmente
- 🔍 Filtrar por **Todas**, **Pendentes** ou **Concluídas**
- 🧹 Limpar todas as tarefas concluídas de uma vez
- 💾 Persistência via `localStorage` (as tarefas são mantidas ao recarregar a página)

## Como usar

Abra o arquivo `index.html` em qualquer navegador moderno — nenhuma dependência ou instalação é necessária.

## Estrutura do projeto

```
Projeto-01/
├── index.html   # Estrutura da página
├── style.css    # Estilos e layout
└── script.js    # Lógica da aplicação
```
